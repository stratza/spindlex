# API Reference

This section provides detailed technical documentation for all modules in SpindleX.

## Core Modules

*   [**Client**](client.md): SSH and SFTP client implementations.
*   [**Server**](server.md): SSH and SFTP server implementations.
*   [**Transport**](transport.md): Low-level SSH transport layer.
*   [**Protocol**](protocol.md): SSH and SFTP protocol message definitions.

## Supporting Modules

*   [**Crypto**](crypto.md): Cryptographic backends and primitives.
*   [**Host Keys**](hostkeys.md): Host key storage and verification policies.
*   [**Logging**](logging.md): Built-in logging and sanitization.
*   [**Tools**](tools.md): Key generation and benchmarking tools.
*   [**Exceptions**](exceptions.md): Exception hierarchy.
